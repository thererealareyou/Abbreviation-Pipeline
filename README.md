# "Автоматический извлекатель доменных аббревиатур (АИДА)"

---

Пайплайн для получения **расфшировки аббревиатур** и **определений терминов** из текста.
Проект нацелен на **дообучение эмбеддинговой модели** для RAG-систем и помощи при реализации **Micro-RAG** с помощью **Query Expansion**.

---

### Особенности
*   **Стек:** Python 3.13, Pandas, llama-cpp.
*   **LLM:** Выбирается на усмотрение пользователя. На видеокартах до 6 GB VRAM включительно рекомендуется использовать **T-lite-it-1.0 (Q4_K_M)** (промпты адаптированы именно под неё).
*   **Работа в четыре этапа:** Разделение задач на 4 этапа для уменьшения галлюцинаций модели и дополнительной проверки, с сохранением всех промежуточных результатов.
*   **Настраиваемость**: В config/settings.yaml можно настроить промпты и параметры под свою задачу.

---

### Запуск

```
git clone <путь_до_репозитория>
cd <путь_до_репозитория>
docker-compose up --build
```

Доступ будет доступен с помощью Swagger (/docs).
